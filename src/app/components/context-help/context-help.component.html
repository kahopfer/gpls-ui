Create a New User

  1. Select the "Create a New User" button from the user management screen to load the "Add a New User" form.
  2. Complete all required fields
  3. Click the "Save" button

Sign a Student In

After logging into the web application your screen will default to the "Sign In" screen. It is here that you can select a student to sign in.

  1. Select one or more students from the table.
  2. Click the green "Sign In" button located at the bottom of the screen.

Sign In Details

After you click "Sign In" the next screen you will see will contain a drop down box for each student signed in.

  1. Select the person who is signing in the student. If all students are being signed in by an employee, select the available check box.
  2. Click the "Sign In" button to complete the sign in process.

Sign a Student Out

To sign out a student choose the "Sign Out" option at the top menu bar. This will then take you to the Sign Out screen. This screen only shows what students are currently signed in.

  1. Select one, multiple or all students from the list.
  2. Click the green "Sign Out" button at the bottom of the screen.

Sign Out Details

The next screen will list all the students selected with a drop down box, sign boxes for extra items and a notes field.

  1. Select who is signing out the student from the drop down box. If the student is being signed out by a staff member, click the available check box.
  2. Select extra items from the provided checklist and add notes if necessary.*
  3. Click the "Sign Out" button at the bottom of the screen again.

NOTE: The sign boxes and notes section are optional fields and do not need to be filled to continue.

Add Line Item

Click the "Add new line item" button from the Create Invoice screen to display a dialog box.

  1. Complete all required fields.
  2. Click the "Save" button to add the new line item.
  3. Click the 'x' in the top right-hand corner of the dialog box to cancel the operation and return to the previous screen.
Modify an Existing Line Item

Modify an existing line item by clicking on the desired row from the Line Items table. The current details for the line item will display in a dialog window.

  1. Make any desired changes to the line item details.
  2. Click the "Save" button to add the new line item.
  3. Click the 'x' in the top right-hand corner of the dialog box to cancel the operation and return to the previous screen.
  4. Click the "Delete" button to permanently delete the line item.

Enroll a New Family

Select the "Enroll a New Family" button from the Families screen to open the Enroll a New Family form.

  1. Enter the Family Name into the provided field.
  2. Complete the required student information.
  3. To add multiple students, select "Add another student +."
  4. Complete the required information for the guardian. To add additional guardians, select "Add another guardian +."
  5. Select the "Save" button to add the family to the system.

Note: Families must contain at least one guardian and one student.

Deactivate and Re-activate

Under the family details screen students and their guardians are listed as either "Active" or "Inactive". Inactive families are not visible on the system.

Deactivate a Family
  1. From the family details screen, click the "Deactivate This Family" button.
  2. Select "Yes" to deactivate a family.
  3. Select "No" to cancel the operation and return to the previous screen.
Deactivate a Student
  1. From the Student Details screen, click the "Deactivate" button.
  2. Select "Yes" to deactivate a family.
  3. Select "No" to cancel the operation and return to the previous screen.
Re-activate a Student
  1. Select the student you wish to re-activate from the Inactive Students tab
  2. From the Student Details screen, click the "Reactivate" button.
  3. Select "Yes" to re-activate the student.
  4. Select "No" to cancel the operation and return to the previous screen.
Re-activate a Guardian
  1. Select the guardian you wish to re-activate from the Inactive Guardians tab.
  2. From the Guardian Details screen, click the "Reactivate" button.
  3. Select "Yes" to re-activate the guardian.
  4. Select "No" to cancel the operation and return to the previous screen.
Delete a Family
  1. Select the "Delete This Family" button from the family details page.
  2. Select "Yes" to delete the family.
  3. Select "No" to cancel the operation and return to the previous screen.

Family Details

The family details page will show a list of all active and inactive students that belong to this family. It will list their first and last names as well as a notes section for all students available to this family. Below that is a Guardians section. In the guardians section you will see a list of all guardians that are in the system to pickup a student for this family. This section will list First and last name, relation to the student and primary and secondary phone numbers.

Manage Families

The Families screen allows administrators to enroll, modify, deactivate, re-activate, and delete families. The administrator's version of the Families screen is identical to the regular user's version, with the exception of the "Enroll a New Family" button. Select Families from the menu bar to view a list of currently enrolled families. From the Families screen click on the name of the family you wish to edit.

View Families

Click on Families in the menu bar at the top of the screen to view family information. On this page the families are listed alphabetically by last name. Here you can click on one family and it will take you to the family details page.

Invoices

To download a .pdf copy of an invoice, select the option button that corresponds with the desired invoice and click the "Download Selected Invoice" button. The name of the downloaded invoice will follow the format [family name]_mm_dd_yyyy.pdf (ex. Cliff_11_16_2017.pdf)

To mark an invoice as paid, select the option that corresponds with the desired invoice and select the "Mark Selected Invoice as Paid" button. Once an invoice has been marked as "paid" the operation cannot be undone. If this is done in error, the invoice must be deleted and regenerated from the Create Invoice screen.

To delete an invoice, select the option button that corresponds with the desired invoice and select the "Delete Selected Invoice" button. This action is also permanent and cannot be undone.

A dialog box will appear to confirm the delete request. Select "Yes" to complete the operation, or "No" to cancel the operation and return to the previous screen.

Invoice List

Invoices

Generated invoices will be available for download from the Invoices screen. The Invoices screen contains two tabs: View all Invoices and View by Family. The View all Invoices tab displays a list of all invoices that have been generated in the system. This view displays the family for which the invoice was generated, the amount charged, the date the invoice was generated, and whether or not the balance was paid. Invoices can be sorted by date in ascending or descending order. Additionally, there is a Paid filter to display only invoices that have been paid (true) or remain unpaid (false).

To download a .pdf copy of an invoice, select the option button that corresponds with the desired invoice and click the "Download Selected Invoice" button. The name of the downloaded invoice will follow the format [family name]_mm_dd_yyyy.pdf (ex. Cliff_11_16_2017.pdf)

To mark an invoice as paid, select the option that corresponds with the desired invoice and select the "Mark Selected Invoice as Paid" button. Once an invoice has been marked as "paid" the operation cannot be undone. If this is done in error, the invoice must be deleted and regenerated from the Create Invoice screen.

To delete an invoice, select the option button that corresponds with the desired invoice and select the "Delete Selected Invoice" button. This action is also permanent and cannot be undone.

A dialog box will appear to confirm the delete request. Select "Yes" to complete the operation, or "No" to cancel the operation and return to the previous screen.

Manage Rates

The Manage Rates screen allows Administrators to edit child care rates and add extra items.

Modify a Childcare Rate
  1. Select the row you wish to change from the Childcare Rates table.
  2. Change the Item Value.
  3. Click the "Save" button to update the rate information.
  4. Click the 'x' in the top right-hand corner of the dialog box to cancel the operation.
Extra Items
  1. Select the row you wish to change from the Extra Items table.
  2. Change the Item Value.
  3. Click the "Save" button to update the rate information.
  4. Click the 'x' in the top right-hand corner of the dialog box to cancel the operation.
  5. Select the "Delete" button from the Rate Information window.
  6. Select "Yes" to delete the rate.
  7. Select "No" to cancel the operation and return to the previous screen.

Reset a User's Password

  1. Select the "Reset Password for Selected User" button from the user management screen to load the "Reset Password" form.
  2. Verify that the listed username belongs to the user whose password needs to be reset.
  3. Enter the new password into the available field.
  4. Click the "Reset Password" button.

Manage Users

The user management screen allows Administrators to add users to the system, delete users from the system, and reset passwords for existing users. To access the user management screen, click the navigation menu and select "Manage Users."

Delete a User
  1. Select the "Delete Selected User" button from the user management screen to load the "Delete User Confirmation" dialog box.
  2. To permanently delete a user from the system, select yes.
  3. To cancel the operation and return to the previous screen, select no.

To view your profile:

A success message should appear near the top right-hand corner.
